## Topics Covered

### Prompt Injection

Manipulating AI behavior by inserting malicious or adversarial instructions into prompts.

### Jailbreaking

Bypassing AI safety mechanisms using roleplay, contextual manipulation, or carefully crafted prompts.

### Prompt Leakage

Extracting hidden instructions, system prompts, or confidential internal information.

### Data Exfiltration

Getting AI systems to reveal information they were explicitly instructed to protect.
